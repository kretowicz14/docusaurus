# Modbus

![Modbus](/img/modbus.png)

Modbus component to use with modbus devices.

## Example config

```yaml title="Example config"
modbus:
  uart: uart1
```

## Configuration variables:

- **uart** (**Required**, string) - UART to define eg uart1, uart2. boneIO currently uses only uart1.
